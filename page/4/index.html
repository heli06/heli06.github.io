JAVA学习笔记（七）

零碎知识

  • 需要命令行的程序，在IDE中可以用run as类似的来运行。
  • 关于java- -X非标准参数：-Xms1024m，设置Java堆最小大小。-Xmx2000m，最大大小。-Xss256k，栈的大小。栈不用太大（不过递归层数多了最好设置得大一点）。因为Java的所有对象和局部变量都在堆中，栈中没多少（这点和C++不同）。
  • System这个类值得关注。System.out是static且final的。
  • 软件设计有许多原则，可以百度。比较著名的有里氏替换原则等。
  • user.home ， usr.dir：程序当前目录所在路径。

Object

函数：protected Object clone()

//这个Student的例子后面还用
//若没有implement，运行时会抛出异常
public calss Student implement Cloneable{
    String id, name;
    int age;
    boolean gender;

    public static void main(String[] a) {
        Student s1 = new Student();
        s1.id = new String("001");
        s1.name = new String("zs");
        s1.age = 18;
        s1.gender = true;

        //(Student)不可少, clone是protected
        //clone实现的是浅复制
        Student s2 = (Student) s1.clone();

    }
}
