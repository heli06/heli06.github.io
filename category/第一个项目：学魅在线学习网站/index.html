第一个项目：学魅在线学习网站

  • 项目

大一小学期，我有幸跟着大三的廖志杰大神学长做了学魅在线学习网站。我做的事情其实很少，主要就是折腾首页在移动端的页面适配。

项目结构在学长的博客上都有，我再偷个懒吧

其实我负责的内容很少，主要是首页的移动端适配。首页顶部的搜索框在手机页面下会隐藏只留下一个按钮，点击按钮后搜索框下滑移出；电脑正常大小的屏幕一行显示四门课程，随着页面的缩小会逐渐变成三行、两行直至一行。
以下这一段是搜索框的动画效果的CSS代码：

.am-topbar-form {
position: absolute;
left: 20%;
margin: 9px 0;
height: 38px;
line-height: 32px !important;
  }


.am-topbar-form-hide {
z-index: -1;
top: 0;
opacity: 0;
transition: all 0.5s;
}


.am-topbar-form-show {
z-index: 1;
top: 56px;
opacity: 1;
transition: all 0.5s;
}
