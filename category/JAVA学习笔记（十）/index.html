GUI编程

使用Java进行GUI编程，有这些库能用：awt，swing，swt，JavaFX。
进行GUI设计可以采用直接拖组件的方式，不必非要直接写代码。
GUI编程里广泛使用了事件注册与监听机制，比较典型的是焦点的聚焦与失焦。这里不多赘述。
任何组件的外观是由一个函数画出来的。

awt

在老师给的代码里有示例程序，路径：\7\GUI。示例程序很简单且自带注释，这里就不赘述了。

GUI编程的最简单的类库。

Component是一切组件类的根类。组件包括Canvas，Button，Checkbox等9个（真的好少）。这些组件是能够改写的。

awt调用的是本地的组件，功能很弱，无法制作外观复杂的软件。而且各个平台上显示的效果不同。

Canvas（画布）是一个更基础的组件类型。如果要一个没有提供的组件，可以用Canvas画出来。

swing

swing拥有丰富的组件，可移植性100%。缺点是运行较慢，因为所有组件都自己绘制。
老师下节课会详细讲。

swt

swt介于awt和swing之间。它调用本地的东西来绘制。
这个老师好像不打算讲。

JavaFX

JavaFX在所有方面大大超越它的前辈们。除了体积较大几乎没有缺点。
老师好像也不打算讲，但我强烈推荐学习一下。
