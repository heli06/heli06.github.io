Tag组件

回顾一下，目前学过的组件有：Servlet，Filter，Listener。

JSP不像HTML和XML这些纯粹的标记语言，这使得JSP写的网页看起来很混乱，可能同时有HTML、CSS、JavaScript、Java的代码；为了让JSP看起来更像标记语言一点，就有了Tag。它可以用来自定义标签。这不仅使得网页看起来简洁，也便于网页的组件化。

Tag在javax.servlet.jsp.tagext包里。接下来介绍几个重要函数（按调用顺序排列）：

setPageContext(PageContext pc)//初始化函数
int doStartTag() throws JspException
int dpEndTag() throws JspException
release//之所以叫`release()`而不是`destroy()`，是因为并不真正销毁对象，只是放回对象池，提高虚拟机效率。

要使用以上Tag，得在标记库描述符文件（.tld）里注明。通常使用工具生成，一般放在WEB-INF下。

.tld文件的解释和标签在JSP文件里的使用方法，可以参考菜鸟教程-自定义标签。注意自定义标签也可以有属性，包括颜色、格式等等。

以后可以去开源网站上找自己想要用的Tag，不必都自己写。
