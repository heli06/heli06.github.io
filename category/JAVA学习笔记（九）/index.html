Hashtable 和 HashMap 的差别

它们的差别在前文的笔记里有提到过：

Map map1 = new HashMap();
Map map2 = new HashTable();

map1.put("001", null);//正确
map1.put("001", null);//错误

若涉及多线程编程，还有一个不能忽略的差别：HashTable是同步的集合，HashMap是未同步的集合。所以HashTable比HashMap更为安全。

当然，HashMap也可以转换为同步的形式：

map = Collections.synchronizedMap(map);
