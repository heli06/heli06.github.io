零碎知识

  • <script>标签放到</body>标签之前能使浏览器更快地加载页面
  • js有关联数组的写法，但不推荐使用——Array对象的属性最好不要随意更改
  • 尽可能使用全等操作符
  • 处理兼容性问题时，尽可能使用对象兼容而非浏览器嗅探技术
  • DOM core比HTML-DOM更冗长，但支持处理所有符合DOM规范的文档
  • 在HTML5中，<acronym>已被<abbr>标签代替
  • CSS样式属性的名字用连字符，DOM里则是驼峰命名法
  • DOM里样式属性的单位不总是与CSS样式表里的设置相同
  • 第253页form对象的有关描述（很长就不贴到这里了）
  • 不要用return false阻止event的默认行为（可参考但别认死理）
  • CSS中强大的EM
  • html移动端页面、图片多少宽度最合适？
