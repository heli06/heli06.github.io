JAVA学习笔记（三）

  • 技术研究

零碎知识

  1. 移位运算向下取整，普通运算向零取整
  2. 如果不想让其它人或程序调用对象，用abstract修饰类,或用privat e修饰构造函数。详见单例模式的例子。
  3. 元素之间可以byte->int,但是数组之间不可以，详见数组的例子。
  4. 覆盖函数的访问权限不能低于被覆盖函数的访问权限
  5. java容易被反编译，找到序列号的类后可以破解或用新写的类代替。可以用混淆程序处理来防止被破解。

接下来是一道很重要很经典的题目

//问以下程序运行后输出什么
//答案: 0, p, 5
//解析见以下注释，按序号顺序查看
class A {
    A() {this.init(); }
    public void init() {// 3.该函数被覆盖
        System.out.println("q");
    }
}
class B extends A {
    public int i = 4;
    B(int i) { // 2.A构造函数先于B构造函数运行
        System.out.println("p");// 6.输出p
        this.i = i; // 7. i被赋值为5
    }
    public void init() {// 4.实际执行的函数
        // 5.此时 i 尚未初始化，为0
        System.out.println(i);
    }
    public static void main(String[] args) {
        // 1.程序开始运行
        System.out.println(new B(5).i);
        // 8.输出
    }       
}

抽象函数和抽象类

在系统早期设计阶段，用抽象函数勾勒出大概框架，不关注实现

abstract class Shape {//有抽象函数的类肯定是抽象类
    abstract public double area();//抽象函数
}
private double w, h;
class Rectangle extends Shape {
    public double area(){
        return w*h;
    }
}
class Circle extends Shape {
    private double r;
    public double area() {
        return Math.PI * r * r;
    }
}

接口(interface)

interface是一个类型，也能用来描述对象

//接口中的函数不能有实现，都是公开的、抽象的
//接口类似于抽象类，但两者是不同的概念
//接口中可以定义数据（必须是常量），但不鼓励
public interface K {
    int i = 7;//相当于public static final int i = 7;
    abstract void a();//加不加abstract都一样
    void b();
}

//类中必须实现接口K的所有函数，否则为抽象类
class B extends Object implements K {
    public void a() {}
    public void b() {}
    public void c() {}
}

K k = new B();
k.a();//正确
k.b();//正确
k.c();//错误，接口没有c函数
(B)k.c();//错误，注意优先级
((B)k).c();//正确

//接口支持多继承
public interface K extends M,N {
    …………
}
class B extends Object implements K, M {
    …………
}

//一个接口的引用变量可以轻易转换为其它的类、接口
Object o = new A();
if(o instanceof K) {
    K k = (K)0;
}

包及打包命令

命名空间：为了防止命名冲突

//类A在包a中
package a;
class A {

}
//其它程序里通过 a.A a1;或者 import a.*;来访问
//*是通配符。

//但如果发生以下情况
import java.sql.*;//有Date
import java.util.*;//也有Date
class Hello {
    java.util.Date d1;//只能用全名
}
//注意以上的import中，包也有层次结构和命名规范。
//通配符只能导入包中的类，不能导入包中的包。
//在一个包中只能有一个公开类。文件名和公开类名相同。

有关的命令

//没有使用别的包时
javac -d d:\ HelloWord.java
java -classpath d:\ HelloWord
// 以下命令会依次搜寻列举的文件夹
java -classpath d:\;c:\;d:\java HelloWord

//使用了别的包时，计算器的例子
//假设文件里写有package com.abc.calc，
//那么执行以下命令时自动映射成文件夹 com/abc/calc

//引用了别的包，所以名字和classpath都不能省略

打包命令: jar

//cvf: 创建一个归档文件
