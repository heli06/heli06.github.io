函数：boolean equals(Object o)

//这是原本自带的实现，但很鸡肋
public boolean equals(Object o) {
    return this == o;
}
//所以要这样覆盖：
public boolean equals(Object o) {
    if(o instanceof Student) {
        Student s = (Student) = o;
        return this.age == s.age &&
            this.gender == s.gender &&
            this.name.equals(s.name) &&
            this.id.equals(s.id);
    }
}

泛型的应用与Collection接口

Collection接口为线性表、向量、栈、队列、优先队列以及集合定义了共同的操作。也就是说，接口List、Set、Map均继承了Collection接口
函数极多，多练多记

Collection c = …………………………;

c.add(new Integer(5));//正确

//早期JDK错误，5不是对象
//更新的版本能自动封箱
c.add(5);

c.add("5");

c.contains("5");//正确
c.contains(new Integer(5));//正确

//c的一切操作都是按值比对
c.remove(new Integer(5));//正确

//[0],则有多少个元素就创建多长的数组
//若超长（大于c的容量）则补充null

泛型提高了精确性，却不够灵活

//传什么都可以，但灵活过头
void a(List list) {

}

//只能传Object,Student不可以
void a(List<Object> list) {

}
//适配所有，却倒退了
void a(List<?> list) {

//受限通配，在精确与灵活间平衡
void a(List<? extends A> list) {

//下限通配，在精确与灵活间平衡
void a(List<? super A> list) {

}

集合（Set）

Set有HashSet、LinkedHashSet、TreeSet来创建集合
Set中的方法与常量与Collection一样，但规定Set的实例不包含重复元素（Set在概念上接近数学中的集合）
HashSet不能排序，LinkedHashSet支持排序，TreeSet是升序的

Set set = new TreeSet();
set.add(new Long(5));
set.add(new Integer(6));//报错，不同类型无法排序
System.out.println(set.size());

iterator(迭代器)

Iterator是一种经典的设计模式，用于在不需要暴露数据是如何保存在数据结构的细节的情况下，来便利一个数据结构。
常用方法：hasNext(),next()

Collection c;
Iterator it = c.iterator();
while(it.hasNext()){
    Object e = it.next();
}

以下是遍历Set的例子

Set set = new HashSet();

//如果这样写，遍历时不需要进行强制类型转换
//Set<String> set = new HashSet<String>();

set.add("hello");
set.add("hello");
set.add("nihao");
Iterator it = set.iterator();
while(it.hasNext()){
    String s = (String) it.next();

    //若之前指定了String，则不需要转换
    //String s =  it.next();

    System.out.println(s);
}
//以下是更简洁的遍历写法
for(Iterator<String> it = set.iterator;;it.hasNext();)
    System.out.println(it.next());

以下是遍历Map的例子

Map<String, Integer> map = new HashMap<String, Integer>();

map.put("001", new Integer());

Set<String> set = map.keySet();//获取“键”的集合
for(Iterator<String> it = set.iterator();it.hasNext();){
    String key = it.next();//获取键
    Integer value = map.get(key);//根据键获取值
    System.out.println(value);//输出值
}
