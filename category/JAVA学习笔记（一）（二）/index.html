（一）组合、封装、多态与继承

先说继承。

class person{
string name;
int age;
}
class Student extends Person{
string student_id;
}

JAVA的类只支持单继承，但接口支持多继承（以后的课会提到）。
以下的内容都与继承联系紧密。

1.组合与聚合

“组合”与“聚合”这两个词都用于描述类与类之间的关系。
聚合(aggregation)关系: 聚合是整体和个体之间的关系,即has-a的关系，此时整体与部分之间是可分离的，他们可以具有各自的生命周期，部分可以属于多个整体对象，也可以为多个整体对象共享；比如人与他买的汽车、公司与员工的关系等。
组合(合成)关系(composition): 他体现的是一种contains-a的关系，这种关系比聚合更强，也称为强聚合；他同样体现整体与部分间的关系，但此时整体与部分是不可分的，整体的生命周期结束也就意味着部分的生命周期结束；比如你和你的大脑。

以下是组合与聚合的例子：

class Car{}
Class Head{}
Class Person{
Car c; //此为聚合，c仅仅是一个引用变量
Head h = new Head();//此为组合
}
public static void main{
Person p = new p();//人一出生就有了脑子，但还没车
Car c1 = new Car();
p.c = c1;//直到这一步才有了车
}

2.封装

通常，一个类的数据成员不能直接被修改，而是通过一对set、get函数来写、读（不一定两者皆有）。
以下是封装的例子：

class student{
    private int age;
    public int getAge(){
        return age;
    }
    public void setAge(int age){
        this.age = age;
    }
}

3.多态

多态在代码层面上有两种体现形式：覆盖（override）与重载（overload）。
此为覆盖的例子：

class A{
    int work(){
        Return 0;
    }
}
class B extends A{
    int work(){
        return 1;
        //若新建一个B的对象b，那么b.work()返回1
    }
}

此为重载的例子：

class adder{
    int add(int i,int j){return i+j;}
    double add(double i,double j){return i+j;}
}

（五）类型转换

可在基本数据类型之间、对象与对象之间转换。但不能在对象与基本数据类型之间进行转换。基本同C++，用法
另外老师上课时举了个挺有趣的例子：

class A{}
class B extends A{}
public static void main(){
    A a = new A();
    B b = new B();
    a = b;//正确
    b = a;//错误，类的类型可向下兼容但反之不可
}

另外一个例子。注意Object是所有类的根类

class A{}
class B extends A{}
public static void main(){
Object o1 = new A();
Object o2 = new B();
Object o3 = new int[3];
A a = new A();
B b = new B();

o1 = o2; //正确
o2 = o1;//正确
o1 = o3;//正确
o3 = a; //正确
o3 = b; //正确
b = (B)o3;//强制类型转换。编译时通过，运行时报错。
}
