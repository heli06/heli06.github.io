RequestDispatcher 中的 forward 和 include

阅读后文前建议先看看这篇：Servlet请求转发之RequestDispatcher

forward的特点：

RequestDispatcher rd = rd.forward(request, responce);
//后面不要继续处理该请求，因为已经转发给别的组件
//req， res的生命周期由其他组件决定

include的特点：

RequestDispatcher rd = rd.include(request, responce);
//后面将继续处理该请求
//req， res的生命周期由本组件决定
